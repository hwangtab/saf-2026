## Minjung Art Aesthetic Implementation (Oh Yoon Special Exhibition)

- **Core Concept**: Woodcut print aesthetic (Minjung Art style)
- **Key Elements**:
  - **Typography**: `font-serif` for traditional feel, high contrast `text-charcoal`.
  - **Borders**: Thick (`border-4`), rough/stamped look using CSS shadows (`shadow-[4px_4px_0px_0px_rgba(...)]`).
  - **Colors**:
    - Background: `bg-canvas-soft` (Hanji/paper feel).
    - Accents: `text-brand-orange` (intense, revolutionary feel).
  - **Decorations**: "Stamped" text boxes, rotated elements (`rotate-1`), double borders (`border-double`).

- **Rationale**:
  - Mimic the rough, hand-carved texture of woodcut prints without using heavy image assets.
  - Maintain accessibility (high contrast) while evoking a specific historical and artistic mood.
