## 인터뷰 질문 템플릿 (10문항)

아래 질문 중 6~8개를 선별하여 사용. 작가 특성에 따라 변형 가능.

### 기본 질문 (필수 3개)

1. **작업 세계 소개**: "작가님의 작업을 한 문장으로 설명한다면?"
2. **영감과 소재**: "어디서 영감을 받으시나요? 최근 작업의 출발점은?"
3. **SAF 참여 동기**: "씨앗페에 참여하신 이유는 무엇인가요? 동료 예술인 상호부조에 대해 어떻게 생각하시나요?"

### 선택 질문 (3~5개 선택)

4. **작업 과정**: "하루의 작업 루틴은 어떻게 되나요? 한 작품을 완성하는 데 보통 얼마나 걸리나요?"
5. **재료와 기법**: "주로 사용하시는 재료나 기법이 있다면? 그 선택의 이유는?"
6. **전환점**: "작가로서 가장 큰 전환점이 된 순간이 있다면?"
7. **컬렉터에게**: "첫 작품을 고민하는 분에게 한 마디 해주신다면?"
8. **금융 차별 경험**: "예술인으로서 금융 서비스를 이용하면서 어려웠던 경험이 있으신가요?" _(민감 질문 — 작가가 원할 때만)_
9. **향후 계획**: "앞으로의 작업 계획이나 목표가 있으신가요?"
10. **추천 작품**: "씨앗페에 출품한 작품 중 특별히 애착이 가는 작품이 있다면?"

## 작가 선정 기준

1. **작품 다양성**: 매체/장르가 다양하도록 배치 (회화 → 판화 → 사진 → 조각 순환)
2. **인기 작품 보유**: 조회수/관심 높은 작품의 작가 우선
3. **스토리 잠재력**: 특별한 작업 배경이나 SAF 참여 동기가 있는 작가
4. **이미지 품질**: 프로필 사진 또는 작업 과정 사진 확보 가능한 작가

### 1개월차 추천 인터뷰 대상 선정 방법

```sql
-- Supabase에서 작품 수 많은 + 미판매 작품 있는 작가 조회
SELECT artist, COUNT(*) as artwork_count,
       COUNT(*) FILTER (WHERE NOT sold) as available_count
FROM artworks
WHERE is_published = true
GROUP BY artist
HAVING COUNT(*) >= 2 AND COUNT(*) FILTER (WHERE NOT sold) > 0
ORDER BY artwork_count DESC
LIMIT 10;
```

→ 이 결과에서 프로필(history) 데이터가 풍부한 작가 2명을 첫 달 인터뷰 대상으로 선정.

## 인터뷰 진행 프로세스

1. **작가 선정** → 위 SQL로 후보 추출
2. **질문지 전달** → 위 템플릿에서 6~8개 선택, 이메일/카카오톡으로 전달
3. **답변 수령** → 서면 또는 구두 (녹음 → 텍스트 변환)
4. **글 작성** → 위 마크다운 구조로 정리 (2,000자+)
5. **작가 확인** → 발행 전 내용 확인 요청
6. **발행** → `/admin/content/stories`에서 입력, `is_published: true`
7. **내부 링크 확인** → 작가 갤러리 페이지 ↔ 인터뷰 글 상호 링크
