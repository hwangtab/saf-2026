### 패턴 C — 정보형 패자 (임프레션만, 클릭 없음)

일반 미술 정보 콘텐츠 → 높은 임프레션 → 낮은 CTR → 실질 유입 없음.

| 페이지                                             | GSC 임프레션 | GSC 클릭 | CTR      | 평균 순위 |
| -------------------------------------------------- | ------------ | -------- | -------- | --------- |
| `/stories/editions-explained`                      | **676**      | 2        | **0.3%** | 7.2       |
| `/stories/reading-art-sizes-ho-vs-cm`              | **249**      | 1        | **0.4%** | 4.8       |
| `/stories/prints-vs-originals-and-edition-numbers` | **137**      | 1        | **0.6%** | 7.9       |

**이것이 "정보 정리형 패자 패턴"이다.** 676 임프레션에도 CTR 0.3% — 사람들은 SERP에서 스니펫으로 답을 얻고 클릭하지 않는다. 어떤 미술 사이트나 쓸 수 있는 "에디션 뜻", "호수 계산법", "판화 vs 원화" 정보는 이미 더 강한 경쟁자가 SERP를 지배하고 있다.

**예외 — "재료 + 씨앗페 작가 연결"형**: `archival-pigment-print-photography`는 정보형이지만 77 imp에 3.4% CTR. 아키벌 피그먼트 프린트 기법을 실제로 사용하는 씨앗페 작가들과 연결된 덕분에 차별점이 생겼다.

## 9. 프레이밍 가드레일 (콘텐츠 제작 시 필수)

> CLAUDE.md §캠페인구조에서 발췌 + 콘텐츠 전략 관점 보완

**출품 작가 = 연대자 (절대 피해자 아님)**  
씨앗페 출품 작가들은 동료 예술인의 금융 차별 문제를 해결하기 위해 **자발적으로 참여한 연대자**다.  
✅ "이미 자리 잡은 작가가 동료를 돕기 위해 작품을 내놓았다"  
❌ "대출 못 받는 불우한 작가들이 모인 페스티벌"

**금융 차별 데이터 = 한국 예술인 전체의 구조적 문제**  
84.9% 배제율은 출품 작가 개인의 상황이 아니라, 한국 예술인 전체를 향한 금융 시스템의 구조적 차별을 다루는 것.

**올바른 흐름 강조**  
"출품 작가들이 작품을 내놓음 → 작품 판매 수익이 상호부조 기금 → 금융 차별을 겪는 예술인에게 저금리 대출"

**숫자 표기**  
작가 수는 절대 하드코딩 금지. i18n `{artistCount}` 변수 또는 `ARTIST_COUNT` 참조.
