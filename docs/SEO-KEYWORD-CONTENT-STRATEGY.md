## 2. 콘텐츠 사일로 구조

```
                    /artworks (갤러리)
                   /          \
          /artworks/category    /artworks/artist
                |                     |
        [카테고리 랜딩]          [작가 프로필]
                                      |
                              /stories (매거진)
                            /     |      \
              artist-story  buying-guide  art-knowledge
                   |              |              |
            [작가 인터뷰]    [구매 가이드]    [미술 지식]
```

**교차 링크 규칙** (이미 구현됨):

- 작가 인터뷰 → 해당 작가의 작품 갤러리 링크
- 구매 가이드 → 관련 카테고리 갤러리 링크
- 작품 상세 → 해당 작가 인터뷰 (있으면)
- 미술 지식 → 관련 카테고리 갤러리

## 4. 콘텐츠 작성 가이드라인

### SEO 최적화 체크리스트

1. **제목**: 타겟 키워드 포함, 55자 이내, 클릭 유도형
2. **excerpt**: 핵심 키워드 자연 삽입, 160자 이내 (meta description으로 사용됨)
3. **body (마크다운)**:
   - H2/H3 소제목에 관련 키워드 배치
   - 첫 200자 이내에 주요 키워드 1회 등장
   - 3,000자 이상 (구글 롱폼 선호)
   - 이미지 3개 이상 (alt 텍스트 필수)
   - 내부 링크 3개 이상 (작품 갤러리, 작가 페이지, 관련 글)
4. **thumbnail**: 1200x630 이상, 글 주제를 대표하는 이미지
5. **tags**: 관련 키워드 3-5개
6. **slug**: 영문 소문자, 하이픈 구분 (예: `first-time-buying-art`)

### 마크다운 내부 링크 패턴

```markdown
작품 갤러리에서 [회화 카테고리 둘러보기](/artworks/category/회화)를 통해 직접 확인해보세요.

이 작가의 다른 작품은 [작가 갤러리](/artworks/artist/홍길동)에서 볼 수 있습니다.
```

> 주의: 마크다운 내부 링크는 locale prefix 없이 작성 (/artworks/...). MarkdownRenderer가 자동 처리.
