# SEO 백링크 회수 및 도메인 권위 빌딩 가이드

작성일: 2026-04-27
대상: SAF 운영진 / 비개발자도 따라할 수 있도록 작성

---

## 왜 이 가이드가 필요한가

GSC(Google Search Console) 3개월 데이터(2026-01~04)에서 확인된 사실:

- **노출되는 검색어 364개 중 95% 이상이 브랜드 검색** ("씨앗페", "씨앗페2026", 작가 이름)
- "그림 구매", "미술작품 구매" 같은 일반 상업 키워드 **노출 자체가 거의 0**
- /artworks 페이지: 노출 662 / 클릭 7 / **CTR 1.06%**
- 한국 외 트래픽: 미국 1,538 노출 / 일본 272 노출 (수요는 있지만 영문은 noindex)

**진단**: 사이트 자체의 SEO(메타·구조·콘텐츠)는 양호. 진짜 문제는 **도메인 권위 부족**. 즉 외부 사이트가 saf2026.com을 충분히 추천(링크)하지 않아 구글이 이 도메인을 "미술품 분야의 신뢰할 만한 출처"로 인정하지 않고 있음.

이 가이드는 **개발 작업이 아닌, 운영자가 직접 실행해야 하는 외부 신호 강화 작업**을 다룹니다.

## 작업 1: 언론 보도 32건 백링크 회수 🔥

### 배경

`/news` 페이지에 정리된 32건의 언론 보도(한겨레·경향신문·KBS·오마이뉴스 등)는 **씨앗페 캠페인을 인용한 기사**들입니다. 그러나 많은 기사들이 **saf2026.com URL을 본문에 링크하지 않은 채** 단순 인용에 그쳤을 가능성이 높습니다.

언론 기사에서 saf2026.com으로의 백링크 1건은 일반 블로그 백링크 50~100건과 맞먹는 가치를 가집니다(언론사 도메인 권위가 높기 때문).

### 단계별 작업

#### Step 1 — 현황 파악 (1~2시간)

`/news` 또는 Supabase `news` 테이블에 등록된 32건 기사를 다음 형식의 스프레드시트로 정리:

| 번호 | 매체   | 제목                | URL         | 발행일     | 기자 이름 | 백링크 유무 | 비고        |
| ---- | ------ | ------------------- | ----------- | ---------- | --------- | ----------- | ----------- |
| 1    | 한겨레 | "동료 예술인 위해…" | https://... | 2026-XX-XX | 김기자    | ❌ 없음     | 본문 인용만 |
| 2    | 경향   | "씨앗페가 묻는다…"  | https://... | 2026-XX-XX | 이기자    | ✅ 있음     | OK          |
| ...  |        |                     |             |            |           |             |             |

**확인 방법**:

1. 각 기사 URL을 열어서 본문에 `saf2026.com`이 포함되어 있는지 검색 (Cmd+F)
2. 포함되어 있으면 ✅, 없으면 ❌
3. ❌인 경우 기자 이름과 매체 연락처를 기록

도구: Google Sheets 또는 Notion 데이터베이스 추천.

#### Step 2 — 정정·추가 요청 메일 작성 (2~3시간)

❌인 기사(아마 20~25건 정도)에 대해 정중한 정정 요청 메일을 작성합니다.

**메일 템플릿 (한국어)**:

```
제목: [씨앗페] 2026.XX.XX 보도 관련 정정 요청 — 공식 사이트 링크 추가 부탁드립니다

[기자 이름] 기자님께,

안녕하세요. SAF(Seed Art Festival, 씨앗페) 2026 캠페인을 운영하는 [이름]입니다.

지난 [발행일] [매체]에 실어주신 "[기사 제목]" 기사에 대해
독자들의 정확한 정보 접근을 위해 한 가지 정정을 부탁드리려 메일 드립니다.

기사 본문에 씨앗페가 언급되어 있으나 공식 캠페인 사이트로의 링크가
포함되어 있지 않아, 관심을 가진 독자들이 캠페인 페이지를 직접 찾기 어려운 상황입니다.

가능하시다면 다음과 같이 본문에 공식 사이트 링크를 추가해 주실 수 있을지요.

- 공식 사이트: https://www.saf2026.com
- 작품 보기/참여하기: https://www.saf2026.com/artworks
- 캠페인 소개: https://www.saf2026.com/about

번거로우시겠지만 검토 부탁드립니다. 추가 정보가 필요하시면
언제든 회신 주시면 됩니다.

감사합니다.

[이름]
SAF 운영진
contact@kosmart.org
```

**핵심 원칙**:

- 정중하게, 강요 없이 — "수정해주세요"가 아니라 "검토해주실 수 있을지요"
- 독자 입장에서의 가치를 강조 — "독자들이 캠페인을 찾기 어렵다"
- 구체적 URL을 정확히 제시 — 기자가 복사해서 바로 쓸 수 있게
- 정정·반박이 아닌 보완임을 명확히

#### Step 3 — 발송 및 추적 (1주)

- 매체별로 한꺼번에 보내지 말고, 하루 5~10건씩 분산 발송
- 발송 후 1주 안에 응답이 없으면 1회 정중하게 follow-up
- 추가 응답 없으면 더는 push하지 않음 (관계 손상 방지)

**예상 회수율**: 20~25건 중 30~50% 응답 → 백링크 6~12건 추가

### Step 4 — 추적 시스템 구축

회수에 성공한 백링크를 추적하기 위해:

1. [Ahrefs Webmaster Tools](https://ahrefs.com/webmaster-tools) 무료 등록 (백링크 모니터링)
2. 또는 GSC의 "링크" 메뉴에서 매주 확인
3. 새 백링크가 추가될 때마다 위 스프레드시트에 ✅ 표시

## 작업 2: 참여 작가 127명 SNS 백링크 요청 🎯

### 배경

127명 작가는 **이 캠페인의 가장 강력한 자연 옹호자**입니다. 각자 인스타그램·블로그·작가 홈페이지를 운영하며, 평균적으로 1~3개의 도메인을 가지고 있습니다.

작가 1명당 백링크 1개만 받아도 **127개 도메인의 추천**이 되며, 이는 도메인 권위에 매우 큰 영향을 줍니다.

### 단계별 작업

#### Step 1 — 작가 연락처 정리 (1일)

이미 `2026 씨앗페 참가자 정보` 스프레드시트가 있다면 그것을 활용. 없다면 다음 정보를 수집:

| 작가 이름 | 인스타그램 | 개인 홈페이지 | 이메일 | 카카오톡 |
| --------- | ---------- | ------------- | ------ | -------- |

#### Step 2 — 작가용 백링크 키트 제작 (반나절)

작가들이 쉽게 백링크를 걸 수 있도록 다음 자료를 패키지로 준비합니다.

**a. 인스타그램·SNS 프로필용 한 줄**

```
🎨 SAF 2026 (씨앗페) 참여 작가 — 동료 예술인을 돕는 상호부조 캠페인
saf2026.com/artworks/artist/[작가이름]
```

**b. 블로그·홈페이지용 배너 이미지 (3종)**

- "SAF 2026 참여 작가" 정사각형 배너 (1080x1080)
- "씨앗페 출품작 보기" 가로 배너 (1200x400)
- "동료 예술인을 돕는 캠페인" 세로 배너 (600x800)

배너 이미지는 작가가 다운로드해서 본인 사이트에 임베드할 수 있게 제공.

**c. HTML 임베드 코드** (홈페이지가 있는 작가용)

```html
<!-- SAF 2026 참여 작가 배지 -->
<a
  href="https://www.saf2026.com/artworks/artist/[작가이름]"
  target="_blank"
  rel="noopener"
  style="display:inline-block; padding:12px 20px; background:#XXX;"
>
  🎨 SAF 2026 참여 작가 페이지 보기
</a>
```

#### Step 3 — 일괄 안내 메일 발송

127명에게 동일한 안내 메일을 발송. 단, 이메일 주소는 BCC로 두어 개인정보 노출 방지.

**메일 템플릿**:

```
제목: [SAF 2026] 참여 작가님께 — 작가 페이지 SNS 공유 부탁드립니다

[작가 이름] 작가님께,

안녕하세요. SAF 2026 운영팀입니다.

씨앗페 온라인 갤러리(saf2026.com)에 작가님의 작품 페이지가 공개되어
운영되고 있습니다.

📍 작가님 페이지: https://www.saf2026.com/artworks/artist/[이름]
📍 등록 작품 수: [N]점

캠페인이 더 많은 사람에게 닿을 수 있도록, 가능하시다면
작가님의 인스타그램·블로그·개인 홈페이지에 위 링크를 한 번
공유해주실 수 있을지요. 작품 판매로 이어지는 것뿐 아니라,
검색 엔진에서 캠페인이 더 잘 노출되는 데도 큰 도움이 됩니다.

도움이 될 만한 자료를 첨부드립니다:
- 인스타그램 프로필용 한 줄 텍스트
- 블로그·홈페이지용 배너 이미지 3종
- 홈페이지가 있으신 분을 위한 임베드 코드

[다운로드 링크]

감사합니다.

SAF 2026 운영팀
contact@kosmart.org
```

#### Step 4 — 응답 추적

- 일주일 후 follow-up 메일 1회 (응답 없는 작가)
- 카카오톡 단톡방이 있다면 그곳에서도 한 번 안내
- 회수 현황 스프레드시트로 관리

**예상 회수율**: 30~50% — 약 40~60명이 SNS·홈페이지에서 링크 공유

## 작업 3: 공공기관·예술 단체 디렉토리 등재 📚

### 배경

`.go.kr`, `.or.kr` 도메인의 공공기관·예술 단체에서 받는 백링크는 **도메인 권위가 매우 높음**. 1건이 일반 블로그 백링크 30~50건과 맞먹습니다.

### 등재 시도할 기관 리스트

#### 예술 정책·진흥 기관

1. **한국예술인복지재단** (`kawf.kr`) — 예술인 복지 사업 디렉토리
2. **한국문화예술위원회** (`arko.or.kr`) — 예술 단체 디렉토리
3. **예술경영지원센터** (`gokams.or.kr`) — 예술 경영 사례
4. **서울문화재단** (`sfac.or.kr`) — 서울 지역 예술 사업
5. **한국사회적기업진흥원** (`socialenterprise.or.kr`) — 사회적기업 사례

#### 사회혁신·연대 단체

6. **한국사회혁신금융** — 사회 임팩트 사업
7. **사회투자지원재단** — 사회 임팩트 투자
8. **희망제작소** — 사회혁신 사례 데이터베이스

#### 미술 분야

9. **국립현대미술관** (`mmca.go.kr`) — 사회 캠페인 미술 사례
10. **한국미술협회** — 협회 회원 단체 등재
11. **한국미술시장협회** (`art-market.or.kr`)

### 등재 요청 작성 시 강조점

각 기관의 디렉토리 등재 기준은 다르지만, 공통적으로 어필할 수 있는 점:

- "예술인 금융 차별 해소"라는 사회적 목적
- "354건 저금리 대출 / 95% 상환율"이라는 측정 가능한 성과
- "127명 작가 자발적 참여"의 규모
- "2023년 1회, 2026년 2회"의 지속성

## 작업 4: 게스트 포스팅·인터뷰 5건 ✏️

### 배경

다른 플랫폼에 글을 기고하고 그 글에 saf2026.com 링크를 자연스럽게 포함시키는 방식. 직접 컨트롤할 수 있는 백링크입니다.

### 타겟 플랫폼 5곳

#### 사회혁신·임팩트

1. **롤링스톤지(롤링스톤 코리아)** — "사회 임팩트 미술" 주제
2. **빅이슈** — 예술인 사회적 권리 주제
3. **오마이뉴스 시민기자** — 누구나 기고 가능, 사회 캠페인 친화적

#### 미술 전문

4. **월간미술** — 한국 미술 시장 칼럼
5. **퍼블릭아트** — 공공미술·사회 미술 칼럼

### 게스트 포스트 주제 추천 (이미 매거진에 작성된 콘텐츠 활용 가능)

다음 주제는 SAF가 권위 있게 다룰 수 있는 영역:

1. **"한국 예술인의 84.9%가 대출에서 배제되는 이유"** — 데이터 기반 르포
2. **"354명 예술인이 95% 상환한 이유"** — 상호부조 모델 분석
3. **"갤러리 50% 수수료의 정체와 작가 직매의 가능성"** — 미술 시장 칼럼
4. **"미술품 구매가 사회 운동이 될 수 있는가"** — 의식 있는 컬렉터 칼럼
5. **"한국 신진 작가가 살아남는 방법"** — 정책 제안

### 진행 방식

1. 각 매체의 기고 가이드라인 확인 (보통 1500~3000자)
2. 매체 톤에 맞게 편집 (월간미술 = 학술적, 빅이슈 = 따뜻한 르포)
3. 본문 안에 saf2026.com 링크 1~2개 자연스럽게 삽입
4. 발행 후 본인 SNS·뉴스레터로도 공유

**예상 효과**: 5건 발행 시 도메인 권위 +5점, 신규 트래픽 월 100~300명

## 주의 사항 — 절대 하지 말 것

다음은 단기 효과처럼 보이지만 구글의 페널티를 받을 수 있는 **블랙햇 SEO**입니다. 절대 하지 마세요.

- ❌ **백링크 구매** — fiverr, 네이버 카페에서 파는 백링크는 99% 스팸 백링크. 발견되면 도메인 자체에 페널티
- ❌ **PBN(Private Blog Network) 구축** — 자기가 만든 가짜 블로그로 자기 사이트에 링크
- ❌ **댓글·게시판 도배** — 다른 사이트 댓글에 saf2026.com 링크 무차별 살포
- ❌ **상호 링크 거래** — "내 사이트 링크해주면 너희 사이트 링크해줄게" — 구글이 명시적으로 패턴을 감지

위 작업 1~4는 모두 **화이트햇** 방식이며, 시간이 걸리지만 안전합니다.

## 마무리 — 이 가이드의 한계와 다음 단계

이 가이드는 **외부 신호 강화**에 초점을 맞췄습니다. 그러나 SEO는 외부 신호 + 내부 콘텐츠 + 기술적 구현의 종합입니다.

이미 완료된 내부 개선 (참고):

- ✅ /artworks 페이지에 SEO 본문(인트로·카테고리 가이드·FAQ) 추가

- ✅ 매거진 시리즈 5편 신규 작성 (그림 구매 키워드 클러스터)

이 가이드의 작업과 위 내부 개선이 함께 작동할 때 효과가 극대화됩니다.

3개월 후 GSC 데이터를 다시 점검하고, 이 가이드를 업데이트할 예정입니다.
