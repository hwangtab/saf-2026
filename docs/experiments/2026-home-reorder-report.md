## 6. 해석과 불확실성

**왜 비율은 오르고 볼륨은 떨어지나?**

가장 유력한 설명은 홈 방문자 구성 변화다. 전체 방문자(1,335→1,339명)는 동일한데 홈 진입자만 −44% 감소했다. POST 기간에 외부 링크(SNS, 카카오 공유)가 특정 작품 상세나 /artworks 갤러리로 직접 연결되었거나, 재방문 사용자들이 홈을 건너뛰고 직접 내부 페이지로 진입한 것으로 보인다. 이 경우 홈에 남은 방문자들은 평균적으로 더 높은 구매 의향을 가진 cohort일 수 있어 비율이 상승한다.

재배열 효과 자체(작품 그리드를 먼저 보여주는 것)가 비율 개선에 기여했을 가능성도 있지만, 현재 데이터로는 선택 편향과 실제 레이아웃 효과를 분리할 수 없다.

**통계적 노이즈**  
14일 구간, 홈 방문자 125명, purchase_click 9명은 소표본이다. 비율 지표의 표준 오차가 크며, 95% CI로 CONCERN/KEEP 경계가 겹칠 가능성이 있다.

**계절성**  
6월 중순(배포 전)과 6월 하순~7월 초(배포 후)는 계절적 소비 패턴 차이(여름 방학 전후, 미술 구매 성수기/비수기)가 있을 수 있다. 외부 교란요인으로 배제 불가.
