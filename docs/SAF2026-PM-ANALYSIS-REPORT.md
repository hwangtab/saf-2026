## 1. Executive Summary

SAF 2026은 **한국 예술인의 금융 차별 문제**를 해결하기 위한 소셜 캠페인 플랫폼이다. 한국스마트협동조합이 운영하며, 예술인 84.9%가 1금융권에서 배제되는 구조적 문제에 대해 **5% 고정금리 상호부조 대출 펀드**와 **온라인 작품 직거래 갤러리**를 핵심 솔루션으로 제공한다.

### 핵심 성과 지표 (현황)

| 지표               | 현재    | 목표        |
| ------------------ | ------- | ----------- |
| 지원 예술인        | 150+명  | 500명       |
| 대출 상환율        | 95%     | 95%+ 유지   |
| 상호부조 펀드      | ₩1.25억 | ₩10억       |
| 대출 실적          | ₩7억+   | ₩10억+ 역량 |
| 온라인 갤러리 작품 | 100+    | 300+        |

### 9가지 분석에서 도출된 Top 5 전략적 권고

1. **임팩트 바이럴 루프 구축** — 구매 후 임팩트 카드 → SNS 공유 → 신규 유입 (Growth Loop #1)
2. **비전시 기간 콘텐츠 루프** — 월간 작가 스포트라이트 → SEO 유입 → 상시 판매 (Growth Loop #2)
3. **ESG 기업 파트너십 + 기획자 제안서 키트** — 기획자 승수 효과 활용 (Growth Loop #3)
4. **결제 통합 또는 Cafe24 UX 개선** — 구매자 전환율 직결 마찰점 해소
5. **North Star 지표 체계 구축** — "월간 대출 순환 완료 예술인 수" 추적 인프라 구축

## 2. 프로젝트 개요

### 2.1 미션

> 1금융권에서 배제된 예술인에게 동료 신뢰 기반의 저금리 대출과 작품 직거래 채널을 제공하여, 예술 생태계의 금융 자립을 실현한다.

### 2.2 문제 정의

한국 예술인의 금융 배제는 3단계 악순환 구조를 갖는다:

| 1단계: 배제        |     | 2단계: 착취                               |     | 3단계: 파괴                                 |
| ------------------ | --- | ----------------------------------------- | --- | ------------------------------------------- |
| 84.9% 1금융권 거절 | →   | 고금리 대출(15-30%) 의존, 48.6% 사채 노출 | →   | 채무 독촉 → 창작 중단, 전시 취소, 관계 파괴 |

**근본 원인**: 예술인의 불규칙 소득과 신용 이력 부족을 기존 금융 시스템이 수용하지 못함.

### 2.3 솔루션 구조

**SAF 2026 플랫폼**

| 상호부조 대출               | 온라인 갤러리          | 캠페인 스토리텔링                   |
| --------------------------- | ---------------------- | ----------------------------------- |
| 5% 고정금리, 동료 신뢰 심사 | 100+ 작품, 작가 프로필 | 데이터·증언·뉴스, 6개 차트·20+ 증언 |

| ---------------- | -------------- | -------------- |

> **외부 연동**: Cafe24 쇼핑몰 (결제·배송) / 소셜펀치 후원 (기부금 모집)

#### Unique Value Proposition (고유 가치 제안)

> **"예술인의 신용은 작품과 동료의 신뢰로 증명됩니다"**
>
> - 단순 후원이 아닌 **상호부조 순환 구조** — 구매·가입이 대출 펀드로 순환
> - 데이터 기반 **사회 문제 가시화** — "84.9% 배제" 통계로 공감·행동 유도
> - **95% 상환율**이라는 실증된 신뢰 모델

## 5. Value Proposition (가치 제안)

### 5.1 세그먼트별 JTBD 가치 제안

#### 세그먼트 1: 금융 위기 예술인

| 파트            | 내용                                                                                                                       |
| --------------- | -------------------------------------------------------------------------------------------------------------------------- |
| **WHO**         | 1금융권 대출 거절 경험이 있는 한국 예술인 (84.9%)                                                                          |
| **WHY**         | "빚 걱정 없이 다음 전시를 준비하고 싶다" — 합리적 금리 자금 조달 + 작품 판매 채널 + 동료 네트워크                          |
| **WHAT BEFORE** | 1금융권 거절 → 카드론 20%+ → 채무 독촉 → 창작 중단. 정부 지원금은 소액·단기·접근성 낮음                                    |
| **HOW**         | 5% 고정금리 상호부조 대출 (동료 신뢰 심사) / 온라인 갤러리 작품 상시 노출·판매 / 아티스트 대시보드 (작품·프로필 직접 관리) |
| **WHAT AFTER**  | 금리 부담 1/4~1/6 감소 (15-30% → 5%). 창작 몰입 회복 + 온라인 추가 수익원 + 동료 네트워크 소속감                           |
| **ALTS**        | 2금융권(고금리), 지인 차용(관계 부담), 정부 지원(소액·경쟁) → SAF만이 "동료 신뢰 기반 저금리 대출 + 판매 채널" 동시 제공   |

**VP Statement**:

> 금융에서 배제된 예술인을 위해, SAF는 동료 신뢰 기반의 상호부조 금융 플랫폼으로 5% 고정금리 대출과 작품 직거래 채널을 제공합니다. 고금리 대출이나 정부 지원금과 달리, 예술인 스스로가 만든 신뢰 네트워크가 금융 자격을 증명합니다.

>

>
> 김서진 같은 예술인은 5% 금리 자체보다 **"나와 같은 처지의 동료 150명이 있고, 95%가 약속을 지켰다"**는 사실에서 더 큰 동기를 얻는다. 금융 배제의 핵심 고통은 돈이 아니라 **고립과 수치심**이며, 협동조합 가입 자체가 치유의 첫 단계.
>
> **제품 시사점**: 온보딩에서 금리 정보보다 **"함께하는 예술인 수"와 "상환 성공 사례"**를 먼저 보여주면 전환율이 높아질 것.

## 7. 경쟁사 분석

### 7.1 경쟁 구도 개요

SAF는 **"예술인 금융 + 작품 직거래 + 소셜 임팩트"**를 결합한 유일한 플랫폼으로, 기존 경쟁사들이 각각 부분적으로만 커버하는 영역을 통합 제공한다. 직접 경쟁자는 없으나, 간접 경쟁자가 각 기능 영역에서 대안을 제공한다.

### 7.2 경쟁사 프로필

#### 1. 오픈갤러리 — 온라인 미술품 렌탈·구매

| 항목            | 내용                                                                        |
| --------------- | --------------------------------------------------------------------------- |
| **유형**        | 간접 경쟁 (작품 판매 채널)                                                  |
| **핵심 서비스** | 월 구독형 미술품 렌탈(₩3만~/월), 원작 구매                                  |
| **강점**        | 렌탈 모델로 진입 장벽 낮춤, UI/UX 세련, 큐레이션 역량, 누적 거래 ₩100억+    |
| **약점**        | 소셜 임팩트 없음, 작가 금융 지원 없음, 작가 스토리 깊이 부족                |

## 10. North Star Metric & 지표 체계

### 10.1 비즈니스 게임 분류

SAF는 **Transaction Game (거래 게임)**에 해당한다.

| 게임 유형              | 해당 여부 | 근거                                          |
| ---------------------- | --------- | --------------------------------------------- |
| Attention (관심)       | ❌        | 체류 시간이 가치가 아님                       |
| **Transaction (거래)** | **✅**    | 대출 실행, 작품 판매, 조합원 가입이 핵심 가치 |
| Productivity (생산성)  | △         | Exhibitor 포털은 부분적 해당                  |

### 10.2 North Star Metric

> **월간 대출 순환 완료 예술인 수**
>
> = 해당 월에 "대출 → 창작 → 작품 판매 → 상환" 사이클을 1회 이상 완료한 예술인 수

#### 7개 기준 검증

| 기준           | 충족 | 근거                                               |
| -------------- | ---- | -------------------------------------------------- |
| 이해 용이성    | ✅   | "대출 받고 갚은 예술인 몇 명?" — 누구나 이해       |
| 고객 중심      | ✅   | 예술인이 실제 금융 자립을 달성했는지 측정          |
| 지속 가능 가치 | ✅   | 순환 완료 = 재대출 자격 = 장기 관계                |
| 비전 정합      | ✅   | "예술 생태계 금융 자립" 미션의 직접 지표           |
| 정량적         | ✅   | 월간 정수 값으로 추적 가능                         |
| 실행 가능      | ✅   | 대출 심사, 갤러리 판매, 상환 관리 등 운영팀이 영향 |
| 선행 지표      | ✅   | 순환 증가 → 펀드 성장 → 더 많은 대출 → 매출 증가   |

## 11. Growth Loops (성장 루프)

### 11.1 Growth Loop 적합성 평가

| Loop 유형          | SAF 적합도 | 근거                                                 |
| ------------------ | ---------- | ---------------------------------------------------- |
| **Viral Loop**     | ★★★        | 구매 후 임팩트 카드 → SNS 공유 → 신규 유입           |
| **Content Loop**   | ★★★        | 작가 스토리·작품 콘텐츠 → SEO/SNS 유입 → 갤러리 방문 |
| **Referral Loop**  | ★★☆        | 조합원→조합원 추천 (동료 신뢰 기반, 자연스러움)      |
| Usage Loop         | ★☆☆        | 예술인 작품 등록은 공유 가능하나 빈도 낮음           |
| Collaboration Loop | ★☆☆        | Exhibitor 포털 협업 가능하나 사용자 수 제한적        |

### 11.2 Growth Loop #1: 임팩트 바이럴 루프 (최우선)

> **구매 → 임팩트 카드 → SNS 공유 → 신규 방문 → 구매**

| 단계                    | 설명                                                        | 핵심 지표                    |
| ----------------------- | ----------------------------------------------------------- | ---------------------------- |
| **1. 구매**             | 구매자가 Cafe24에서 작품 구매 완료                          | 월간 판매 건수               |
| **2. 임팩트 카드 수신** | 구매 확인과 함께 "당신의 구매가 기여한 효과" 카드 자동 생성 | 카드 생성률                  |
| **3. SNS 공유**         | 작품 사진 + 임팩트 카드를 인스타/페이스북에 공유            | 공유율 (목표: 구매자의 20%+) |
| **4. 친구 유입**        | 공유물을 본 친구/팔로워가 SAF 갤러리 방문                   | 공유당 클릭 수 (목표: 5+)    |
| **5. 재구매/신규 구매** | 방문자가 작품에 관심 → 구매                                 | 공유 유입 → 구매 전환율      |

**루프 계수 추정**:

- 월간 판매 10건 × SNS 공유율 20% = 2건 공유
- 공유당 클릭 5회 × 구매 전환율 2% = 0.2건 추가 판매
- **루프 계수: 0.02** (초기, 최적화 필요)

**실행 방안**:

| 항목              | 내용                                       |
| ----------------- | ------------------------------------------ |
| Phase 1 (0-1개월) | 임팩트 카드 디자인 + 공유 버튼 구현        |
| Phase 2 (1-2개월) | SNS 공유 템플릿 + #SAF2026 해시태그 캠페인 |
| Phase 3 (2-3개월) | 공유 인센티브 (공유 시 다음 구매 5% 할인)  |
| 측정              | 공유율, 공유당 유입 수, 유입→구매 전환율   |

### 11.3 Growth Loop #2: 콘텐츠 루프 (장기 성장 엔진)

> **작가 스토리 콘텐츠 → SEO/SNS 유입 → 갤러리 방문 → 구매 → 더 많은 작가 참여 → 더 많은 콘텐츠**

| 단계                 | 설명                                                         | 핵심 지표           |
| -------------------- | ------------------------------------------------------------ | ------------------- |
| **1. 콘텐츠 생산**   | 월간 작가 스포트라이트, 창작 과정 스토리, 캠페인 데이터 기사 | 월간 콘텐츠 발행 수 |
| **2. 검색/SNS 유입** | 구글 검색(SEO) + SNS 노출 → 갤러리 방문                      | 오가닉 트래픽       |
| **3. 갤러리 탐색**   | 콘텐츠 → 작품 상세 → 다른 작품 브라우징                      | 세션당 페이지뷰     |
| **4. 구매/후원**     | 작품 구매 또는 소셜펀치 후원                                 | 전환율              |
| **5. 플라이휠 강화** | 판매 성공 → 더 많은 작가 참여 의향 → 더 많은 콘텐츠 소재     | 신규 작가 가입 수   |

**실행 방안**:

| 항목        | 내용                                                                        |
| ----------- | --------------------------------------------------------------------------- |
| 콘텐츠 유형 | 월간 작가 인터뷰, 창작 비하인드, "Our Reality" 데이터 업데이트, 구매자 후기 |
| 채널        | 블로그(SEO), 인스타그램, 페이스북, 카카오톡 채널                            |
| 주기        | 주 1회 콘텐츠 발행 (월 4건)                                                 |
| KPI         | 오가닉 트래픽 월 30%+ 성장, 콘텐츠 → 갤러리 전환율 10%+                     |

### 11.4 Growth Loop #3: 기획자 승수 루프 (B2B 확장)

> **기획자 SAF 전시 → 성공 사례 → 다른 기업에 제안 → 신규 전시 → 더 많은 작가 노출 → 더 많은 판매**

| 단계                    | 설명                                      | 핵심 지표           |
| ----------------------- | ----------------------------------------- | ------------------- |
| **1. 기획자 온보딩**    | 전시 기획자가 Exhibitor 포털 가입         | 기획자 등록 수      |
| **2. 첫 전시 성공**     | SAF 협업 전시 1건 개최 + 성과 데이터 확보 | 전시 개최 수        |
| **3. 사례 재활용**      | 성공 데이터로 다른 기업에 동일 포맷 제안  | 제안서 발송 수      |
| **4. 신규 전시**        | 기업 ESG 전시 수주 → SAF 작가 참여        | 기업 전시 수        |
| **5. 작가·구매자 확대** | 전시 → 작가 노출 → 판매 → 신규 작가 유입  | 전시 기반 판매 건수 |

**승수 효과**: 1명 기획자 → 연 3-5건 기업 전시 → 작가 20-50명 노출 → 판매 10-30건

**실행 방안**:

| 항목    | 내용                                                                              |
| ------- | --------------------------------------------------------------------------------- |
| Phase 1 | Exhibitor 포털에 "기업 제안서 키트" 탑재 (PPT 템플릿, 성과 데이터, 작가 카탈로그) |
| Phase 2 | SAF 2026 전시 성과 데이터 패키징 (방문자·판매·미디어 노출)                        |
| Phase 3 | 기획자 대상 워크숍 (연 2회) — "ESG 전시 기획법"                                   |
| KPI     | 기획자 포털 활성 사용자 수, 기업 전시 수주 건수, 전시당 매출                      |

| **필요 리소스** | A안: 결제 시스템 개발 (높음), B안: UX 개선 (낮음)                                 |
| **측정 지표**   | 갤러리→구매 전환율, 이탈률                                                        |
| **실행 난이도** | A안: 🔴 높음, B안: 🟢 낮음                                                        |
| **기대 효과**   | 구매 전환율 향상, 구매 데이터 통합                                                |

#### 권고 4: 예술인 온보딩 UX 개선 (소속감 우선) 🟢 보완

**근거**: Persona 1 Insight ("소속감 > 금리") + Value Prop 온보딩 문구

| 항목            | 내용                                                                                         |
| --------------- | -------------------------------------------------------------------------------------------- |
| **목표**        | 예술인 가입 → 활성화 전환율 개선                                                             |
| **실행 방안**   | 온보딩 플로우에서 "함께하는 예술인 150명", "95% 상환 성공 사례" 먼저 노출. 금리 정보는 2순위 |
| **필요 리소스** | 온보딩 페이지 UX 수정                                                                        |
| **측정 지표**   | 가입→조합원 전환율, 작품 등록 완료율                                                         |
| **실행 난이도** | 🟢 낮음                                                                                      |
| **기대 효과**   | 예술인 이탈률 감소, 소속감 기반 리텐션                                                       |

#### 권고 5: 구매자 임팩트 체감 기능 🟢 보완

**근거**: Persona 2 미충족 니즈 (임팩트 카드) + Value Prop (구매=기여)

| 항목            | 내용                                                                                            |
| --------------- | ----------------------------------------------------------------------------------------------- |
| **목표**        | 구매자 재구매율 20%+ 달성                                                                       |
| **실행 방안**   | 구매 완료 후 "임팩트 카드" 표시 (이 구매가 기여한 펀드 금액, 지원된 예술인 수). SNS 공유 최적화 |
| **필요 리소스** | 임팩트 카드 디자인·개발                                                                         |
| **측정 지표**   | 재구매율, SNS 공유 횟수                                                                         |
| **실행 난이도** | 🟢 낮음                                                                                         |
| **기대 효과**   | 재구매 동기 강화, 바이럴 루프                                                                   |
