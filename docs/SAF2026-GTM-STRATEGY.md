## 3. 채널별 실행 플랜

### 🔴 채널 1 — 인스타그램 릴스 (작가 스토리 시리즈)

**목표**: 브랜드 인지도 → 작품 페이지 클릭
**포맷**: 60~90초, 작가 1명 = 릴스 1편
**콘텐츠 구조**:

```
훅 (0~3초):   "저는 동료를 위해 그림을 내놨습니다"
문제 (3~15초): 예술인 금융 차별 데이터 1개 시각화
연대 (15~40초): 작가가 왜 참여했는지 본인 목소리
CTA (40~60초): "지금 SAF에서 작품을 만나보세요"
```

**운영 리듬**: 주 2~3편 (110명 작가 = 37주 치 콘텐츠)
**KPI**: 릴스 조회수 / 프로필 방문 / 링크 클릭

---

### 🔴 채널 2 — PR / 언론 보도

**목표**: 신뢰도 확보 + 무료 대규모 노출
**타깃 매체**:

| 매체 유형        | 구체적 타깃          | 앵글                       |
| ---------------- | -------------------- | -------------------------- |
| 문화/예술 전문지 | 아트인컬처, 월간미술 | 작가 연대 캠페인의 의미    |
| 사회혁신 미디어  | 임팩트온, 더나은미래 | 상호부조 금융 모델         |
| 경제지           | 한국경제, 매일경제   | 아트테크 + 임팩트 투자     |
| 방송             | KBS/MBC 문화 코너    | 84.9% 데이터 + 인간 스토리 |

**보도자료 패키지 구성**:

- 핵심 통계 인포그래픽
- 작가 대표 사진 + 한 줄 증언
- 2023→2026 성과 비교표
- 인터뷰 가능 작가 명단

---

### 🔴 채널 3 — B2B CSR 직접 영업

**목표**: 건당 고단가 구매 + 기업 후원 확보
**타깃**: 문화재단 보유 기업 CSR팀, 예술 후원 이력 있는 기업
**제안 패키지**:

```
[SAF 기업 임팩트 패키지]
✅ 작품 다량 구매 (사무실 설치 또는 임직원 선물)
✅ ESG 보고서 활용 가능한 임팩트 리포트 제공
✅ 기업 로고 → SAF 웹사이트 후원사 노출
✅ 참여 작가와의 기업 아트 워크숍 (옵션)
✅ 보도자료에 공동 후원사 명기
```

**접근 경로**: LinkedIn 담당자 직접 DM + 한국사회적기업진흥원 네트워크 활용

---

### 🟡 채널 4 — 카카오채널 + 카카오톡 공유

**목표**: 한국 고연령 컬렉터 도달 (인스타 약 계층)
**운영**:

- 카카오채널 개설 → 작품 소개 카드뉴스 주 1회
- 각 작품 페이지에 "카카오톡 공유" 버튼 최적화
- "이 작품, 지인에게 선물로 어떠세요?" 메시지 추가

---

### 🟡 채널 5 — 뉴스레터 (구매자 리텐션)

**목표**: 1회 구매자 → 반복 구매자 전환
**발송 주기**: 분기 1회
**콘텐츠**:

```
제목: "당신이 산 그림이 만든 변화 — 1분기 임팩트 리포트"
본문: 기금 조성액 / 대출 집행 건수 / 수혜 작가 익명 사연 / 신규 추천 작품
CTA: "다음 작품 보러가기"
```

## 4. 런치 플랜 (90일 로드맵)

### Phase 1 — 기반 구축 (월 1, 1~30일)

```
Week 1-2: 기금 투명성 대시보드 페이지 완성
Week 2-3: 보도자료 + 작가 스토리 릴스 15편 사전 제작
Week 3-4: B2B CSR 제안서 템플릿 + 타깃 기업 리스트 50개 작성
Week 4:   소프트 런치 — 작가/지인 네트워크 한정 공유, 초기 피드백 수집
```

### Phase 2 — 런치 & 확산 (월 2, 31~60일)

```
Day 31:    공식 런치 — 보도자료 배포 + 릴스 집중 업로드 (주 5편)
Day 32-40: 언론 인터뷰 응대 + SNS 공유 유도 캠페인
Day 41-50: B2B 기업 미팅 집중 진행 (주 5개사 이상)
Day 51-60: 중간 성과 집계 → 반응 좋은 콘텐츠 포맷 2배 투자
```

### Phase 3 — 최적화 & 글로벌 (월 3, 61~90일)

```
Day 61-70: i18n 영문 완성 → 해외 K-아트 커뮤니티 공략 시작
Day 71-80: 1차 구매자 뉴스레터 발송 + 재구매 캠페인
Day 81-90: 90일 성과 리뷰 → 다음 분기 전략 수정
```

## 총괄 진단 요약

**서비스 진단**: SAF는 구조적 차별점(순환 상호부조 모델)과 강력한 데이터 내러티브를 보유한 고잠재력 플랫폼입니다. 현재의 핵심 병목은 "인지도"와 "기금 투명성"이며, 이 두 가지를 해소하면 바이럴 전파와 B2B 고단가 전환이 동시에 가능합니다.

**홍보 방안 진단**: 예산 제약 조건 하에서 **작가 스토리 릴스 + PR + B2B 직접 영업**의 3트랙이 최고의 ROI를 제공합니다. 유료 광고보다 진정성 있는 콘텐츠가 이 캠페인의 본질에 더 부합하며, 실제로 더 효과적입니다.
